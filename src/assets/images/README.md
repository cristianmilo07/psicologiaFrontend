## Nota
Si no tienes las imágenes, puedes:
1. Usar imágenes de stock gratuitas (Unsplash, Pexels)
2. Crear ilustraciones simples con herramientas como Canva
3. Usar iconos de Flaticon o Font Awesome
4. Generar imágenes con IA (DALL-E, Midjourney, etc.)

Las imágenes deben ser en formato PNG con fondo transparente para mejor integración.
